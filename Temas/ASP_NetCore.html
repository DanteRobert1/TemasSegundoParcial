¿QUE ES?

ASP.NET ES UN MARCO DE DESARROLLO WEB POPULAR PARA COMPILAR APLICACIONES WEB EN LA PLATAFORMA WEB.

RENDIMIENTO

EN LAS PRUEBAS COMPARATIVAS INDEPENDIENTES, ES MÁS RÁPIDO QUE OTROS MARCOS WEB POPULARES.

APLICACIONES

LAS APLICACIONES EN LA QUE SE PUEDEN DESARROLLAR Y EJECUTAR SON EN WINDOWS, LINUX, MACOS Y DOCKER.

CARACTERISTICAS

MODERNO

PERMITIR QUE LOS COMPONENTES EN TIEMPO DE EJECUCIÓN, LAS API, LOS COMPILADORES Y LOS LENGUAJES EVOLUCIONEN RÁPIDAMENTE, A LA VEZ QUE PROPORCIONA UNA PLATAFORMA ESTABLE Y COMPATIBLE PARA MANTENER LAS APLICACIONES EN EJECUCIÓN.

IMPLEMENTACION FLEXIBLE

EL ENTORNO DE EJECUCIÓN PUEDE IMPLEMENTARSE COMO PARTE DE LA APLICACIÓN O INSTALARSE DE FORMA CENTRALIZADA EN EL SERVIDOR WEB.
