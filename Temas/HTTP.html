Tecnologías de Web API

HTTTP

“PROTOCOLO DE TRANSFERENCIA DE HIPER TEXTO” (HYPER TEXT TRANSFER PROTOCOL)

PERMITE LA NAVEGACIÓN WEB, YA QUE PERMITE LA TRANSFERENCIA DE DATOS ENTRE LOS CLIENTES (COMO LOS NAVEGADORES WEB) Y LOS SERVIDORES (DONDE SE ALMACENAN Y SIRVEN LOS SITIOS WEB Y SUS CONTENIDOS).

IMPORTANCIA DEL HTT

ES ESENCIAL PARA EL FUNCIONAMIENTO DE LA WEB MODERNA, FACILITANDO LA TRANSFERENCIA DE DATOS Y LA INTEROPERABILIDAD ENTRE CLIENTES Y SERVIDORES, Y PROPORCIONANDO UNA BASE PARA EL DESARROLLO SEGURO DE APLICACIONES Y SITIOS WEB.

URI

SIRVE PARA IDENTIFICAR O LOCALIZAR LA LOCALIZACIÓN DE UN RECURSO. QUE PUEDE SER UNA URL QUE INDICA CÓMO ACCEDER AL RECURSO O UNA URN QUE PROPORCIONA UN NOMBRE ÚNICO PARA EL RECURSO.

URL

IDENTIFICA UN RECURSO INDICADO DONDE ESTA Y COMO ALCANZARLO

CODIGO DE ESTADOS

UN CÓDIGO DE ESTADO INDICA COMO VA LA PETICIÓN, SI TUVO ÉXITO O NO, CAMBIOS, PERMISOS ETC. SIEMPRE ESTA FORMADO POR 3 DÍGITOS Y UNA DESCRIPCIÓN:.
